# Weather App using Python API

## Description
A CLI weather application that fetches live weather data using an API.

## Features
- City-based weather search
- Temperature, humidity, condition
- Uses real-time API

## Tech Used
Python
Requests library
OpenWeatherMap API

## Get start with this

1. Clone the repository
       git clone https://github.com/g00562/Weather-App.git
2. Install the required dependencies
       pip install -r requirements.txt
3. Run 
       python weather.py
